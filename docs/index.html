BISICLES user level documentation

There is a modest collection of user level documentation which aims to show how to build and run BISICLES, and to select run-time options.
  1. Building and running BISICLES
    1. Generic instructions
    2. Site specific notes
  2. BISICLES executables
    1. driver - used to run ice sheet simulations
    2. solverBenchmark - used to run the ice velocity solver from the main code as a standalone solver
    3. control, used to solve BISICLES' inverse problem.
    4. file tools, used to perform miscellaneous actions on BISICLES hdf5 files
  3. Run-time options
    1. Surface fluxes (accumulation and melting)
    2. LevelData interface (used to load meshed data, for example thickness and topography)
    3. Python interface
    4. cdriver interface (used to control BISICLES from an external C or FORTRAN program)
  4. Example applications
    1. Frank Pattyn's MISMIP3D
    2. A Pine Island Glacier example which uses real topography and thickness data
  5. Viewing and analysing data
    1. Viewing data with visit
    2. Analyzing data with R, MATLAB, Python, etc
